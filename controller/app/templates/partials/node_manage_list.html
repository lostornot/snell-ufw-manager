📜 节点初始化脚本

添加节点后，在落地 VPS 上以 root 执行以下命令完成初始化，然后回来点击「测试连接」。

方式一：一键命令（需要落地机能访问控制中心）
curl -fsSL http://{{ ctrl_ip }}:8899/api/nodes/setup-script | bash
方式二：粘贴执行（离线可用）
#!/bin/bash # Snell UFW Manager — 节点初始化 # 请以 root 在落地 VPS 上执行 set -euo pipefail PUBKEY='{{ pubkey }}' CTRL_IP='{{ ctrl_ip }}' echo "══════════════════════════════════════════" echo " Snell UFW Manager — 节点初始化" echo "══════════════════════════════════════════" # 1. 创建 snellmgr 用户 echo -n " 创建 snellmgr 用户... " if id snellmgr &>/dev/null; then echo "已存在 ✓" else useradd -r -m -s /bin/bash snellmgr echo "完成 ✓" fi # 2. 配置 sudoers echo -n " 配置 sudoers... " echo 'snellmgr ALL=(root) NOPASSWD: /usr/local/sbin/snell-fwctl' \ > /etc/sudoers.d/snellmgr chmod 440 /etc/sudoers.d/snellmgr echo "完成 ✓" # 3. 部署 SSH 公钥 echo -n " 部署 SSH 公钥... " mkdir -p /home/snellmgr/.ssh if [ -n "$CTRL_IP" ] && [ "$CTRL_IP" != "<控制中心IP>" ]; then echo "from=\"$CTRL_IP\" $PUBKEY" > /home/snellmgr/.ssh/authorized_keys else echo "$PUBKEY" > /home/snellmgr/.ssh/authorized_keys fi chmod 700 /home/snellmgr/.ssh chmod 600 /home/snellmgr/.ssh/authorized_keys chown -R snellmgr:snellmgr /home/snellmgr/.ssh echo "完成 ✓" # 4. 创建备份目录 echo -n " 创建备份目录... " mkdir -p /opt/snell-fwctl/backups chown -R snellmgr:snellmgr /opt/snell-fwctl echo "完成 ✓" # 5. 启用 UFW 日志 echo -n " 启用 UFW 日志... " ufw logging on 2>/dev/null || true echo "完成 ✓" echo "" echo "══════════════════════════════════════════" echo " ⚠️ 还需要部署 snell-fwctl 脚本" echo " 请使用「方式一」或手动部署" echo "══════════════════════════════════════════"

💡 方式一会自动部署完整的 snell-fwctl 脚本。方式二仅完成用户和权限配置，需要另外部署 snell-fwctl。
